Titel: Sudoku
Spielstart:-Man erstellt ein neues Objekt der Klasse Sudoku, bei dem man eine Nummer eingeben muss.
           -Die Nummer 0 erstellt ein leeres Sudokufeld
           -Die Nummern 1-5 sind einfache Sudokus.
              ->Am Anfang wird immer ein leeres Sudoku erstellt
            
        Hinweise:-Es dürfen nur Zahlen von 0 bis 9 eingefügt werden
                 -Wenn beim Kontrollieren in einem Feld die Zahl 0 steht, gilt das Sudoku als Falsch gelöst
                 -Die Methode "Kontrolle" kann auf jedes Sudoku angewendet werden, auch wenn Sudokunummer 0
                  ist und man alle Zahlen selber eingetragen hat
