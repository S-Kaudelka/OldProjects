Wenn ein Turm Platziert werden soll, zuerst auf den entsprechenden Platz Klicken

    und dann 
        -Q fuer einen Pfeilturm
        -W fuer einen Armbrustturm oder
        -E fuer einen Kanonenturm
           druecken.

Um einen Turm zu verkaufen, R druecken.

Um einen Turm um eine Stufe zu erhoehen, T druecken, maximallevel ist 4,
            danach kann man eine Spezialisierung Auswählen.

Um einen Turm zu Spezialisieren, wenn er Stufe 4 hat, -F drücken für einen schnell-schiessenden Turm
                                                      -G drücken für einen starken Turm
                                                      
Speichern: M (Zwischen den Wellen)
Laden:     N (Nur bei der Levelauswahl)
